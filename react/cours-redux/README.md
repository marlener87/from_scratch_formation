# Cours Redux

** INSTALLER REDUX **
`npm i`
`npm i -s redux react-redux @reduxjs/toolkit @redux-devtools/extension`
Télécharger Redux DevTools

1- Provider qui englobe l'app
2- Création store
3- Création reducers
4- Création actions

** JSON-SERVER **
npm i -g json-server
"json-server --watch db.json" || "npm run server"


** Pour lancer le front **
`npm start`
